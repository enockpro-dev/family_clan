# Clan Book

Hii ni programu ndogo ya Python ya kusimamia taarifa za ukoo wa familia.

## Vipengele

- Kusajili mtu kwenye dirisha la desktop
- Ukurasa wa kwanza wenye chaguo la `Ingia` au `Jisajili`
- Mtumiaji aliyeingia anaweza kutafuta mtu mwingine aliyesajiliwa na kuona uhusiano wao
- Kuhifadhi taarifa kwenye SQLite ili zibaki hata baada ya kufunga programu
- Kuunganisha mtu na baba yake na mama yake
- Bado unaweza kutumia amri za terminal ukihitaji

## Jinsi ya kuendesha programu

```powershell
python clan_book.py
```

Dirisha lina sehemu hizi:

1. Ukurasa wa kwanza unapofungua programu
2. Chaguo la `Ingia` au `Jisajili`
3. `Ingia` hukagua kama mtu tayari amesajiliwa
4. `Jisajili` hufungua fomu ya kuingiza taarifa za mtu
5. Baada ya kuingia, mtumiaji anaweza kutafuta mtu mwingine aliyesajiliwa na kuona uhusiano wao

## Mifano ya amri

Kusajili mtu:

```powershell
python clan_book.py add --name "John Okello" --clan "Lion" --gender "Male"
```

Kusajili mtoto na kuweka jina la baba:

```powershell
python clan_book.py add --name "Peter Okello" --father "John Okello"
```

## Maelezo

- Faili ya database hutengenezwa moja kwa moja kama `clan_book.db`
- Tkinter mara nyingi huja pamoja na Python kwenye Windows
- Fomu ya desktop haiulizi taarifa ya mwaka wa kuzaliwa
- Fomu ya usajili hutumia jina la kwanza, jina la pili, jina la ukoo, jinsia, wazazi, na maelezo
- Mtu anaweza kusajiliwa hata kama wazazi wake bado hawajawekwa kwenye mfumo
- Mzazi akija kusajiliwa baadaye kwa jina linalofanana, programu inaweza kuunganisha uhusiano huo moja kwa moja
- Utafutaji wa uhusiano hufanya kazi vizuri zaidi wakati wanafamilia wamesajiliwa na kuunganishwa kwenye mfumo
